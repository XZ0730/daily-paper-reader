## 摘要
由大型语言模型（LLMs）引导的进化程序搜索已成为自动化科学发现的一种强大范式。然而，当前方法受到三个瓶颈的根本性制约：结构盲目的父代选择、稀疏的整程序评估奖励，以及在搜索过程中无法适应的静态变异算子。我们提出了GAE（图增强进化），这是一个通过紧密耦合的三支柱架构解决这些局限性的框架。首先，关系图神经网络（GNN）将程序解析为带类型的计算图，生成结构感知的嵌入。其次，一个经过强化学习优化的元控制器利用这些嵌入，用有向策略取代盲目进化采样，根据奖励历史动态选择最优父代和变异方向。第三，一个在线GRPO微调循环在测试时使用组归一化的评估奖励持续更新LLM变异算子，直接将模型的生成分布与高适应度的结构性编辑对齐。我们在一项具有挑战性的科学发现任务上评估GAE：针对复杂非线性振荡器系统的符号回归。通过将随机搜索转化为有向、自我改进的轨迹，GAE高效地发现闭合形式的物理方程，始终匹配或超越静态LLM驱动基线，并实现了最先进的分布外性能。

## Abstract
Evolutionary program search guided by Large Language Models (LLMs) has emerged as a powerful paradigm for automated scientific discovery. However, current approaches are fundamentally constrained by three bottlenecks: structurally blind parent selection, sparse whole-program evaluation rewards, and static mutation operators that fail to adapt during search. We present GAE (Graph-Augmented Evolution), a framework that resolves these limitations through a tightly coupled, three-pillar architecture. First, a relational graph neural network (GNN) parses programs into typed computation graphs, producing structure-aware embeddings. Second, an RL-optimized meta-controller leverages these embeddings to replace blind evolutionary sampling with a directed policy, dynamically selecting optimal parents and mutation directions based on reward history. Third, an online GRPO fine-tuning loop continuously updates the LLM mutation operator at test-time using group-normalized evaluation rewards, directly aligning the model's generation distribution with high-fitness structural edits. We evaluate GAE on a challenging scientific discovery task: symbolic regression for complex nonlinear oscillator systems. By transforming stochastic search into a directed, self-improving trajectory, GAE efficiently discovers closed-form physical equations, consistently matching or outperforming static LLM-driven baselines and achieving state-of-the-art out-of-distribution performance.

### 1. 论文的核心问题与整体含义

现有基于大型语言模型的进化程序搜索在自动化科学发现中展现出潜力，但其受到三个相互关联的瓶颈的根本性制约：
*   **奖励稀疏性**：整个程序的评估（如一次完整训练或数值积分）仅返回一个单一的标量信号，大量接近最佳但未严格超越当前最优的程序被丢弃，丢失了有价值的细粒度结构改进信息。
*   **无信息（结构盲目的）的父代选择**：当前方法（如 AlphaEvolve）将程序群体视为一个无结构的集合，仅基于适应度排名和随机多样性进行父代采样，忽略了程序内部承载的丰富结构信息，导致变异行为不可预测，搜索过程趋于随机探索。
*   **静态的变异算子**：作为变异算子的 LLM 在整个搜索过程中其权重被冻结，始终从固定的预训练分布中采样结构变换，无法根据搜索过程中积累的经验来调整变异策略。

因此，该论文的核心目标是提出一种新框架，将随机、盲目的进化搜索过程转化为有向的、自我改进的轨迹，从而更高效地在复杂程序空间中定位高质量区域。

### 2. 论文提出的方法论

GAE 框架通过一个紧密耦合的三大支柱架构来解决上述问题，所有组件都构建在一个共享的**关系图神经网络（GNN）编码器**之上：
*   **核心组件：关系图神经网络编码器**
    *   **程序图构建**：将每个候选程序通过静态抽象语法树分析与成带类型的计算图 \(G = (V, E)\)。节点表示数学运算，边编码数据流。
    *   **图编码**：使用带关系感知的消息传递网络将图嵌入为结构向量 \(z \in \mathbb{R}^d\)。具体地，通过边类型特异的权重矩阵 \(W_r\) 转换源节点特征，进行平均聚合并与自环连接，经过多层图卷积和全局池化（均值、最大值拼接）后得到最终的图级表示 \(z\)。
    *   **在线微调**：在进化过程中，GNN 编码器会定期（每 3 代）在一个滑动窗口上使用均方误差损失（预测其嵌入对应的真实评估分数）进行微调，以跟踪进化中不断变化的程序分布，确保嵌入的时效性。

*   **支柱二：基于强化学习的变异选择（元控制器）**
    *   **状态与动作**：状态是父代程序的 GNN 嵌入 \(z_\alpha\)。动作空间是离散的，例如（变异类型，目标节点 ID，参数索引），其中变异类型包括添加、替换、删除节点，插入常数或子树重写等。
    *   **策略学习**：采用离散 Soft Actor-Critic 算法训练一个策略网络 \(\pi_\phi(a|z)\)。该元控制器根据父代结构选择最可能有产出的变异类型，而非随机选择。
    *   **奖励函数**：一个三组分奖励，仅基于每代生成的最佳子代计算：
        \[
        r = \underbrace{\text{clip}(R(\beta))}_{\text{任务得分}} + \lambda_1 \underbrace{(1 - \min_{p \in P} \cos(z_\beta, z_p))}_{\text{嵌入空间新颖性奖励}} - \lambda_2 \underbrace{\max(\frac{N(\beta)-20}{20}, 0)}_{\text{复杂度惩罚}}
        \]

*   **支柱三：基于 GRPO 的 LLM 在线微调**
    *   **组生成**：每次从父代程序 \(\alpha\) 和一个由 SAC 选择的变异提示 \(a\) 出发，LLM 生成一组 \(G\) 个子代 \(\{\beta_1, \dots, \beta_G\}\)。
    *   **优势计算**：每个子代基于其相对于父代的奖励改善计算组内相对优势 \(R^{\text{grp}}_g = R_{\beta_g} - R_\alpha\)，然后进行组归一化 \(\hat{A}_g\)。
    *   **策略更新**：使用一个 PPO-clip 代理目标函数就地更新 LLM 的权重 \(\theta_{\text{LLM}}\)，损失函数为：
        \[
        \mathcal{L}_{\text{GRPO}}(\theta_{\text{LLM}}) = -\frac{1}{G} \sum_g \min\left( \rho_g \hat{A}_g, \text{clip}(\rho_g, 1 \pm \epsilon_{\text{clip}}) \hat{A}_g \right)
        \]
        其中 \(\rho_g\) 是新旧策略的概率比。这直接使 LLM 的生成分布与高适应度的结构性编辑对齐。

这三个组件优化不同的参数集，不共享梯度，并在不同的时间尺度上运行，保证了模块的独立性和可替换性。
